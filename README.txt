---------------------INSTRUCTIONS FOR USING PDF TO JPG CONVERTER-----------------------------------------------------------------------

For installation:

1.) Open and install "GraphicsMagick-1.3.32-Q16-win64-dll"

2.) Open and install "gs950w64"

3.) Open and install "node-v12.13.0-x64"

4.) Open the "source" folder and open a command line

5.) type <npm install> (without the brackets) and hit Enter


For use:

1.) Drop all pdf files to be converted in "DROP_PDF_HERE" folder

2.) Go into the "source" folder and open a command line

3.) type <node convert> (without the brackets) and hit Enter
